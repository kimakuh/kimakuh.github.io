Final 의 쓰임

자바에서 절대 변하지 않는 특정한 것을 정하고 싶을 때는 Final을 사용
이 키워드는 변수, 메소드, 클래스에 모두 사용할 수 있다.
변수에 사용할 경우 변하지 않는 상수
메소드가 사용할 때는 재정의가 불가능한 메소드
클래스에서 사용할 때는 상속이 불가능한 하나의 완전한 클래스

값이 바뀌는 것을 막는다.

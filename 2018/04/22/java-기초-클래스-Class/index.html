클래스는 객체 지향 프로그래밍에 있어서 가장 기본이 되는 것.
클래스를 이용하여 현실 게계의 특정한 물건을 지칭할 수 있다.
가장 많이 사용되는것이 Node클래스.
개발 프로젝트에서는 하나의 처리할 데이터 단위를 명시하는데 사용되기도 한다.
